=== Rate 'n Tweet ===
Contributors: softerize
Tags: rate, twitter, share
Requires at least: 2.0
Tested up to: 2.9
Stable tag: 1.1

Rate 'n Tweet adds a rating system that forward the user to his Twitter page so the user can share it via Twitter.

== Description ==

Rate 'n Tweet adds a links to Twitter with a automatic rating. You can change the appearence in the
Settings > Rate 'n Tweet menu.

== Installation ==

Upload the Rate 'n Tweet plugin to your blog, activate it, and change the settings as you wish.
